# Changelog

Формат версий: `X.Y.Z.W.Codename` — см. [`src/version.py`](src/version.py).

## [Unreleased] — 2026-05-28

### GUI — темы и визуальный стиль

- Добавлены три режима темы: **Светлая / Тёмная / Авто** (следует системной теме Windows 11).
- Выбор темы определяется через реестр Windows (`AppsUseLightTheme`) без внешних зависимостей.
- Выбор сохраняется в `ui_config.json` и восстанавливается при следующем запуске.
- Кнопки смены темы `☀ Светлая`, `🌙 Тёмная`, `⚙ Авто` размещены в заголовке окна.
- При смене темы UI пересобирается с сохранением всего контекста (папки, кадры, настройки).
- Введены две полные палитры: `dark` (Adobe Photoshop–стиль) и `light` (Windows 11 светлая).

### GUI — ползунки (X / Y / Масштаб / Поворот)

- Ползунки полностью переписаны на custom Canvas вместо `ttk.Scale`.
- Тёмный трек, акцентная заливка активной части, светло-серый круглый thumb — всё чётко видно.
- Поддержка прокрутки колёсиком мыши (Shift × 10 для грубой настройки).
- Thumb и трек не сливаются с фоном ни в светлой, ни в тёмной теме.

### GUI — миниатюры кадров

- Починены галочки выбора кадров для экспорта: непомеченные теперь имеют видимый серый контур.
- Помеченный кадр — синий квадрат с белой галкой; снятый — прозрачный с чётким бордером.

### GUI — прочее

- Фавиконка приложения (`assets/image/favicon.ico`) устанавливается через `iconbitmap`.
- Шапка разделена на titlebar (название + переключатель тем) и toolbar (путь + кнопки действий).
- Убрана кнопка «⬇ Дроп» (временно не работала).
- Панель цветокора: кнопки **Сохранить**, **Сбросить**, **Применить к папке** объединены в одну строку.
- Максимальный масштаб увеличен с 2.00 до 3.00.
- При загрузке новой папки первый подкаталог с исходниками открывается автоматически.

### Сборка (PyInstaller)

- Исправлена ошибка `Importing the numpy C-extensions failed` в собранном exe.
- Заменено `collect_submodules("numpy")` на `collect_all("numpy")` — теперь включаются `.pyd`/`.dll`.
- То же изменение применено для `PIL`.
- Иконка exe задаётся через поле `icon=` в `.spec`-файле.
- Папка `assets/` добавлена в список копируемых директорий при сборке.

### Файлы

- `assets/image/favicon.ico` — иконка окна и exe-файла.
- `assets/image/favicon.png` — PNG-версия иконки.
- `ui_config.json` — хранит последний выбор темы (создаётся при первом запуске).

## [0.0.0.2.ProtoAlpha] — 2026-05-26

### Кадрирование и редактор

- Единая система координат смещения X/Y для превью и экспорта (холст 1400×1050).
- Экспорт рендерит тот же рабочий кадр, что в редакторе.
- Автодетекция товара на превью до 1200 px вместо 360 px.
- Поворот без чёрных углов: берётся дополнительная область исходника, без принудительного зума.
- Смещение X/Y корректно работает после поворота (экранные оси → оси исходника).

### GUI

- Drag-and-drop через `windnd` (без нестабильного WinAPI subclassing).
- Отдельное окно **«Дроп-окно»** для перетаскивания папки/файла.
- Поддержка дропа файла (берётся родительская папка).
