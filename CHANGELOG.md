# Changelog

Формат версий: `X.Y.Z.W.Codename` — см. [`src/version.py`](src/version.py).

## [0.0.0.2.ProtoAlpha] — 2026-05-26

### Кадрирование и редактор

- Единая система координат смещения X/Y для превью и экспорта (холст 1400×1050).
- Экспорт рендерит тот же рабочий кадр, что в редакторе.
- Автодетекция товара на превью до 1200 px вместо 360 px.
- Поворот без чёрных углов: берётся дополнительная область исходника, без принудительного зума.
- Смещение X/Y корректно работает после поворота (экранные оси → оси исходника).

### GUI

- Drag-and-drop через `windnd` (без нестабильного WinAPI subclassing).
- Отдельное окно **«Дроп-окно»** для перетаскивания папки/файла.
- Поддержка дропа файла (берётся родительская папка).

## [0.0.0.1.ProtoAlpha] — 2026-05-26
