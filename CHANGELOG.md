# Changelog

## 2026-05-26

- Улучшено качество JPG-экспорта: уменьшены артефакты и потеря резкости.
- Исправлен рендер кадра при редактировании (без преждевременного проблемного кропа).
- Переработано дерево папок: иерархический вид, корректные имена, стабильный возврат.
- Добавлена панель цветокоррекции (профиль, контрастность, тени, температура, оттенок).
- Настройки цветокора сделаны редактируемыми.
- Добавлена кнопка `Применить ко всем в папке` для массового применения настроек.
- Добавлено управление с клавиатуры: сдвиг кадра стрелками на 1px.
- Добавлена опция `Использовать дроплет` с маршрутизацией кадров по droplet-файлам.
- Снижено количество ложных предупреждений droplet (`code 1` при фактически успешной обработке).
