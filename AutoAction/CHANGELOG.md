## [0.0.1.20.ProtoAlpha] — 2026-06-25

### Режимы прохода

- **Основной** — дроплеты из `droplets/Main/` (сценарий 1→01, 2–4,8→02-03-04-08, 5–7→05-06-07); совместимость с дроплетами в корне `droplets/`.
- **Старый** — все файлы через `droplets/Old/old.exe`.
- Режим задаётся для **папки** (плашка на карточке) или **отдельного файла** (плашка в строке; точка • = свой режим, не как у папки).
- Цветовая индикация: синий — основной, янтарный — старый.
- Скрипт `setup_droplets.bat` — подготовка папок Main/Old.

### Интерфейс

- Компоновка **rail + очередь + command dock** (UX 2026).
- Тёмная шапка окна Windows (DWM).
- Боковая панель: добавление папок, сводка, легенда режимов.
- Нижний dock: прогресс, скорость, кнопка **«Запустить»**.

### Очередь и файлы

- Несколько корневых папок, drag-and-drop (безопасный `safe_dnd` для Python 3.13).
- Карточки папок с раскрытием списка 1.jpg–8.jpg.
- Галочки «в очереди / пропустить»; выбор файлов Ctrl+клик, Ctrl+A.
- Рекурсивный разбор вложенных папок с 1.jpg–8.jpg.

### Обработка

- Пакетный запуск Photoshop-дроплетов с прогрессом и остановкой.
- Проверка наличия дроплетов перед стартом.
- Окно Photoshop сворачивается во время работы дроплета (WinAPI, без COM/JSX).

### Интерфейс (доп.)

- Иконка и логотип из `assets/image` (`favicon.ico`, `favicon.png`).
- Toast по завершении обработки с баннером `MSG_Good.png` (как в основном приложении).
